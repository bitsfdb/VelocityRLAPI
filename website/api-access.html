You don't need an account to use the VelocityRL desktop app. Accounts are only for developers who want higher rate limits on the public REST API.

API Access

Create a free account to unlock 300 requests per minute instead of the default 60.

300 requests / minute

5x the unauthenticated limit. Pass your Bearer token on each request to get the higher quota.

Authenticated identity

Rate limit is tied to your account, not your IP. Useful when sharing an IP with other users.

Same API, same endpoints

No separate keys to manage. Sign in, copy your session token, and add one header to your requests.

Free, no payment required

API access is free. No credit card, no trial, no expiry.
