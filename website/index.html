How it works

Three steps. No external tools. Fully reversible.

01

Set your game folder

Point VelocityRL at your Rocket League CookedPCConsole directory. It reads your installed game files directly.

02

Pick your items

Browse the full item catalog. Choose a source item (what you own) and a donor item (what you want it to look like).

03

Apply the swap

VelocityRL patches the .upk package file for the source item to reference the donor's assets. Launch Rocket League. Your item looks different.

Built to be safe

Every swap is local. Nothing leaves your machine.

Automatic backups

Every original file is backed up before modification. Restore any swap with one click.

Integrity checks

The engine binary is verified against a SHA-256 checksum on every launch. No tampering, no surprises.

Live item database

Item catalog syncs from the VelocityRL API, always up to date with the latest Rocket League content.

UPK-level patching

Operates directly on Unreal Engine 3 package files. Supports bodies, wheels, decals, boosts, and more.

Open source

The full source is on GitHub: app, engine, and API. Audit it, fork it, contribute.

Windows & Linux

Native Windows installer plus Linux AppImage. Nightly builds drop every night from the dev branch.

For developers
